Backend Setup And Configure

Add Multiple Wordpress Sites

Steps To Add Backend:

  • Step 1: Go to wordpress site and install following plug-ins:
    • WooCommerce Plugin by WooThemes
    • WooCommerce Rest API Extension Plugin By TechSpawn
    • WordPress to odoo connector Plugin By TechSpawn
  • Step 2: After installing WooCommerce at Wordpress Site, go to WooCommerce Menu -> Settings -> API -> Keys/Apps then click on Add Key (Note: Permissions are Read/Write)
  • Step 3: Copy generated Consumer Key and Consumer Secret
  • Step 4: Go to odoo site and Install Odoo Woo Connect
  • Step 4: Open ODOO WOO Connect -> WP Site -> Create and then paste the Consumer Key and Consumer Secret
  • Step 5: Check Connection
